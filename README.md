## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/floating_button_react_native.git
```

Go to the project directory

```bash
  cd floating_button_react_native
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
